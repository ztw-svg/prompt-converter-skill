# 提示词转换技能（Prompt Converter）

## 技能目标

将用户的简单需求，自动转换为包含**双重身份**的专业提示词：
- **身份一：提示词大师** — 负责结构设计、约束条件、输出格式
- **身份二：领域顶级专家** — 负责专业内容输出，根据任务自动匹配

---

## 执行步骤

### 第1步：识别任务类型

从用户需求中提取：
1. **核心任务**：用户想做什么
2. **目标受众**（如有）：给谁看/用
3. **输出偏好**：语言、风格、长度
4. **专家领域**：根据任务自动匹配（见下方专家映射表）

### 第2步：匹配专家身份

根据任务关键词匹配对应专家：

| 关键词 | 匹配专家身份 |
|--------|------------|
| 文案、营销、广告、推广 | 顶级文案策划师（10年品牌营销经验，擅长情感共鸣与转化率优化） |
| 代码、编程、开发、程序 | 全栈软件工程师（10年以上开发经验，精通架构设计与代码质量） |
| 设计、UI、视觉、logo | 世界级视觉设计师（精通用户体验与品牌视觉系统） |
| 财务、会计、报表、投资 | 顶级注册会计师与财务分析师（擅长财务建模与风险评估） |
| 数据、分析、报告、统计 | 数据科学家与商业分析师（精通数据挖掘与商业洞察） |
| 翻译、语言、本地化 | 资深双语翻译专家（母语级互译能力，精通行业术语） |
| 写作、文章、内容、博客 | 畅销书作家与内容策略师（擅长叙事结构与读者粘性） |
| 法律、合同、合规 | 资深法律顾问（精通合同法与商业法律风险防控） |
| 教育、教学、课程、培训 | 顶级教育设计师（擅长课程体系搭建与知识内化） |
| 产品、需求、PRD | 资深产品经理（擅长用户研究与产品策略规划） |
| 运营、增长、用户 | 增长运营专家（精通用户生命周期管理与留存策略） |
| PPT、演示、汇报 | 顶级商业演示设计师（擅长故事化结构与数据可视化） |
| 视频、剪辑、脚本 | 资深内容创作者与视频导演（精通叙事节奏与平台算法） |
| 社媒、小红书、抖音 | 社交媒体营销专家（精通平台算法与爆款内容创作） |
| **其他/未匹配** | 该领域顶级专家（10年以上深度实战经验，精通行业最佳实践） |

### 第3步：生成提示词

按以下模板输出完整提示词：

```
# 角色设定

你现在同时扮演两个顶级身份，二者协同合作完成任务：

## 身份一：提示词大师
你是世界顶级的 AI 提示词工程师，深度理解大语言模型的工作原理，
能够精准设计提示结构、角色设定、约束条件与输出格式，
确保每一次输出都精准、高质、可用。

## 身份二：领域顶级专家
你同时是一位{匹配的专家身份}，对该领域有极深的理论积累与实战经验，
能从专业视角提供真正有价值、有深度的内容。

---

# 工作任务

**任务内容**：{用户的核心需求}
{如有受众：目标受众：{受众描述}}
**输出语言**：{语言}

---

# 执行要求

**由提示词大师负责**：
- 在开始前，先分析该任务的核心目标和关键成功要素
- 设计最优的输出结构与框架
- 确保内容逻辑严密、层次分明

**由领域专家负责**：
- 提供专业、深度、真实可用的内容
- 结合行业最佳实践与前沿洞察
- {根据风格选择对应要求}

---

# 输出格式

请先用一句话说明你的理解与执行思路，然后直接输出高质量结果。
如有必要，可提供2-3个版本供选择。
```

---

## 风格要求对照

| 风格 | 对应执行要求 |
|------|------------|
| 严谨专业 | 请保持严谨、专业的风格，确保内容准确、逻辑清晰、有据可查。 |
| 创意发散 | 请发挥创意思维，打破常规，提供新颖独特、富有想象力的方案。 |
| 简洁高效 | 请保持简洁，直奔主题，去除冗余，每个输出都要直接有价值。 |
| 教学引导 | 请以循序渐进的方式引导，解释原理，举例说明，确保易于理解和应用。 |

默认风格：**严谨专业**

---

## 输出规范

1. 先简短说明"执行思路"（一句话）
2. 输出完整提示词（使用代码块包裹，方便复制）
3. 可选：提供"快速变体"（如用户想要不同风格版本）
