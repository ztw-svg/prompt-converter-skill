# 提示词转换技能 · 通用版 (Prompt Converter Skill - Universal)

一个将简单工作需求自动转换为**双重身份专业提示词**的技能，支持所有主流 AI 平台。

## 功能介绍

输入你想做的事情，自动生成包含以下两个身份的专业提示词：

- **提示词大师**：负责结构设计、约束条件、输出格式优化
- **领域顶级专家**：根据任务类型自动匹配（支持 20+ 领域）

生成的提示词可**直接复制到任意 AI 对话框**使用，无需任何修改。

## 使用方法

### 触发词

在任意 AI 平台输入：
- "帮我写提示词"
- "优化提示词"
- "我需要一个提示词让AI帮我做..."
- "提示词转换"

### 示例输入

> 帮我写一个提示词，让AI帮我写产品介绍文案

### 示例输出

生成完整的双重身份专业提示词，末尾标注：
> 💡 此提示词已适配所有主流AI平台，可直接复制使用
