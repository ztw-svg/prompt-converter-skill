# 提示词转换技能 (Prompt Converter Skill)

一个用于 Claude 的 Skill，能将简单的工作需求自动转换为**双重身份专业提示词**。

## 功能介绍

输入你想做的事情，自动生成包含以下两个身份的专业提示词：

- **提示词大师**：负责结构设计、约束条件、输出格式优化
- **领域顶级专家**：根据任务类型自动匹配（文案、编程、设计、财务等20+领域）

## 支持的领域

| 领域 | 匹配专家 |
|------|--------|
| 文案/营销 | 顶级文案策划师 |
| 代码/编程 | 全栈软件工程师 |
| 设计/UI | 世界级视觉设计师 |
| 财务/会计 | 注册会计师与财务分析师 |
| 数据/分析 | 数据科学家与商业分析师 |
| 法律/合规 | 资深法律顾问 |
| 产品/需求 | 资深产品经理 |
| 更多... | 自动匹配领域专家 |

## 使用方法

### 触发词

直接告诉 Claude：
- "帮我写提示词"
- "优化提示词"
- "我需要一个提示词让AI帮我做..."
- "提示词转换"

### 示例

**输入**：帮我写一个提示词，让AI帮我写产品介绍文案

**输出**：完整的双重身份专业提示词，可直接复制使用

## 文件结构

```
prompt-converter-skill/
└── SKILL.md    # 技能主文件
```

## 安装方式

将 `SKILL.md` 文件放入你的 Claude Skills 目录即可使用。
