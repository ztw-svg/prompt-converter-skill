# 提示词转换技能（Prompt Converter）
## 通用版 · 适配所有主流 AI 平台

> 支持平台：Claude / ChatGPT / Gemini / Copilot / 文心一言 / 通义千问 / 讯飞星火 / Grok 等

---

## 技能目标

将用户的简单需求，自动转换为包含**双重身份**的专业提示词：

- **身份一：提示词大师** — 负责结构设计、约束条件、输出格式
- **身份二：领域顶级专家** — 负责专业内容输出，根据任务自动匹配

生成的提示词可直接复制到**任意 AI 对话框**使用，无需修改。

## 通用性设计原则

生成提示词时，需遵守以下原则确保跨平台兼容：

1. **避免平台专属语法**：不使用任何平台独有的指令或格式（如 Claude 的 XML 标签、ChatGPT 的插件调用等）
2. **使用纯文本结构**：用 `#` 标题和 `---` 分隔线组织内容，所有平台均可正确解析
3. **明确角色边界**：每个身份的职责描述清晰，避免模型混淆
4. **语言自适应**：默认中文，用户要求英文时切换，双语任务时并行输出
5. **长度适中**：控制在 500 字以内，避免超出部分平台的上下文限制
